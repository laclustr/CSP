paddle module¶

class paddle.Paddle(playernum)¶

Bases: object

Represents a Paddle object used in the Pong game.

Contains logic for moving paddles, resetting paddles to starting positions, and keeping track of player scores.

playernum¶

Indicates whether this paddle belongs to Player 1 or Player 2.

Type:

int

score¶

The current score of the player.

Type:

int

surf¶

The surface representing the paddle.

Type:

pygame.Surface

rect¶

The rectangle defining the paddle’s position and size.

Type:

pygame.Rect

speed¶

The movement speed of the paddle.

Type:

int

draw(screen)¶

Draws the paddle on the game screen.

update(dt)¶

Updates the paddle’s position based on user input.

reset()¶

Resets the paddle to its initial position.

draw(screen)¶

Draws the paddle on the game screen.

Parameters:

screen (pygame.Surface) – The game screen where the paddle will be drawn.

reset()¶

Resets the paddle to its starting position on the screen. Player 1’s paddle starts on the left, while Player 2’s paddle starts on the right.

update(dt)¶

Updates the paddle’s position based on user input and delta time.

Parameters:

  • dt (int) – The time in milliseconds since the last frame, used to calculate

  • frames. (movement with consistent speed across)
